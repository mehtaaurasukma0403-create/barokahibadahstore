Cara menggunakan aplikasi

  1. Masuk ke halaman Login (untuk simulasi).
  2. Buka menu Transaksi untuk input penjualan.
  3. Isi semua field, lalu klik Simpan Transaksi.
  4. Gunakan tombol Lihat Daftar Transaksi untuk melihat tabel transaksi.
