Profil Aplikasi

Barokah Ibadah Store adalah aplikasi web sederhana untuk mencatat transaksi penjualan barang alat ibadah Islam seperti sajadah, mukena, Al-Qur'an, tasbih, sarung, dan lain-lain.

Fitur yang tersedia

  • Menu navigasi: Login, Beranda, Transaksi, Profil.
  • Form input transaksi penjualan (wajib isi semua field).
  • Daftar transaksi dalam bentuk tabel Bootstrap.
  • Validasi input menggunakan JavaScript (submit ditolak jika data belum lengkap).
